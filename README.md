# 心语桥 — 心智障碍群体沟通表达辅助工具

## 项目背景

沟通是人类最基本的需求之一，但对于心智障碍群体（包括智力障碍、自闭症谱系障碍、唐氏综合征等）而言，**表达自我**却是一项巨大的挑战。

据统计，我国心智障碍人士超过 **1200 万人**，其中相当一部分存在不同程度的**语言表达障碍**：

- **表达困难**：许多心智障碍人士能够理解他人的话语，却难以用语言准确表达自己的想法和需求
- **情绪沟通障碍**：无法有效传达自己的感受（开心、难过、焦虑、疼痛等），容易导致情绪爆发或被误解
- **社交隔阂**：沟通障碍加剧了与社会的隔离，影响人际关系建立和心理健康
- **依赖性强**：日常沟通过度依赖照护者猜测，丧失自主表达的机会和意愿

传统的沟通辅助方案（如纸质图片交换系统 PECS）虽然有效，但存在**携带不便、内容固定、扩展性差**等局限，难以适应多样化的沟通场景。

## 项目意义

心语桥致力于通过**数字化 AAC（增强与替代通讯）技术**，为心智障碍群体搭建一座通往自我表达的桥梁：

### 💬 让每一个想法都被听见

心语桥提供直观的**图文符号界面**，用户只需点击或选择对应的图片/图标，即可快速表达自己的需求、感受和想法，无需依赖复杂的语言能力。

### 🎨 多场景沟通支持

覆盖心智障碍群体日常生活的**核心沟通场景**：

| 场景 | 示例 |
|------|------|
| **基本需求表达** | 我饿了、我想喝水、我想去洗手间 |
| **情绪感受表达** | 我很开心、我很难过、我有点害怕、我身体不舒服 |
| **日常活动选择** | 我想画画、我想听音乐、我想去散步 |
| **社交互动** | 你好、谢谢、对不起、我想和你一起玩 |

### 🧠 降低表达门槛

- **视觉优先**：以图片和图标为主要沟通媒介，降低文字阅读依赖
- **分类清晰**：将沟通内容按场景和类别组织，帮助用户快速定位
- **语音辅助**：选中内容自动朗读，帮助用户确认表达意图，也让沟通对象理解
- **个性化定制**：支持根据用户的具体需求添加自定义内容

### 🤝 促进双向理解

心语桥不仅帮助心智障碍人士**表达自己**，也帮助他们的家人、老师、同事和社区成员**更好地理解**他们，减少因沟通不畅导致的误解和冲突，促进真正的社会融合。

### 📱 随时随地的沟通伙伴

基于移动端设计，用户可以**随身携带**自己的"声音"，在任何需要沟通的场景中自主表达，不再只能被动等待他人询问。

## 适用场景

| 场景 | 说明 |
|------|------|
| **家庭生活** | 帮助心智障碍子女向家人表达日常需求和情感 |
| **特殊教育学校** | 作为课堂沟通辅助工具，促进师生互动 |
| **康复训练机构** | 配合语言治疗和社交技能训练使用 |
| **社区融合活动** | 帮助心智障碍人士参与社区生活，自主沟通 |
| **就业支持** | 在工作场所辅助心智障碍员工与同事交流 |

## 技术特点

- 纯前端应用，零外部依赖，打开即用
- 图文符号 + 语音播报的多模态交互
- 响应式设计，适配手机和平板
- 无障碍设计（大按钮、高对比度、简洁布局）
- 支持个性化内容定制

## 快速开始

直接在浏览器中打开 `index.html` 文件即可使用。
